04

Série avec effectifs (pondérée)

Quand tu as un tableau avec les valeurs en colonne A et leurs effectifs (nombre de fois) en colonne B.

Effectif total
=SOMME(B1:B10) copier
Somme de tous les effectifs nᵢ = n total.
Moyenne pondérée x̄
x̄
=SOMMEPROD(A1:A10;B1:B10)/SOMME(B1:B10) copier
Σ(xᵢ × nᵢ) ÷ Σnᵢ — sans colonne intermédiaire.
Formule mathématique
x̄ = Σ(xᵢ × nᵢ) / Σnᵢ copier
Chaque valeur compte proportionnellement à son effectif.
★
SOMMEPROD calcule A1×B1 + A2×B2 + … + A10×B10 automatiquement. C'est la formule la plus utile pour les séries pondérées.
Exemple — tableau de notes avec effectifs

Notes en A : 8, 10, 12, 14, 16 | Effectifs en B : 2, 5, 8, 4, 1

Effectif total : =SOMME(B1:B5) → n = 20

SOMMEPROD : 8×2 + 10×5 + 12×8 + 14×4 + 16×1 = 16+50+96+56+16 = 234

Moyenne : =SOMMEPROD(A1:A5;B1:B5)/SOMME(B1:B5) → 234 ÷ 20 = 11,7

05

Indicateurs de dispersion

Étendue e
e = max − min
=MAX(A1:A20)-MIN(A1:A20) copier
Simple mais très sensible aux valeurs aberrantes.
Écart interquartile EI
EI = Q₃ − Q₁
=QUARTILE(A1:A20;3)-QUARTILE(A1:A20;1) copier
Couvre 50 % central. Robuste aux valeurs extrêmes.
Écart type σ
σ = √( (1/n)·Σ(xᵢ−x̄)² )
=ECARTYPE.PEARSON(A1:A20) copier
Dispersion globale autour de la moyenne. Plus σ est grand, plus les données sont dispersées.
Variance σ²
σ² = (1/n)·Σ(xᵢ−x̄)²
=VAR.P(A1:A20) copier
Carré de σ. Toujours ≥ 0. Si σ² = 0 → toutes les valeurs sont identiques.
✓
Interpréter σ : si σ est faible → les données sont regroupées près de la moyenne. Si σ est élevé → les données sont très étalées. En général σ < 2 est faible pour des notes sur 20.
!
Ne pas confondre : ECARTYPE.PEARSON (population entière, à utiliser en seconde) ≠ ECARTYPE.STANDARD (échantillon, pour les statistiques avancées). Le premier divise par n, le second par n−1.
06

Exemple complet corrigé

ℹ
Série : notes d'un devoir en A1:A12 → 6, 8, 9, 11, 12, 13, 13, 14, 15, 16, 17, 20
Résultats attendus — à vérifier avec Excel

n (effectif) : =NB(A1:A12) → 12

Somme : =SOMME(A1:A12) → 154

x̄ (moyenne) : =MOYENNE(A1:A12) → ≈ 12,83

Me (médiane) : =MEDIANE(A1:A12) → 12,5 (moy. de la 6e et 7e valeur = (13+12)/2)

Min / Max : =MIN() → 6 | =MAX() → 20

e (étendue) : =MAX()-MIN() → 14

Q₁ : =QUARTILE(A1:A12;1) → 9,75

Q₃ : =QUARTILE(A1:A12;3) → 15,25

EI (interquartile) : Q₃ − Q₁ → 5,5

σ (écart type) : =ECARTYPE.PEARSON(A1:A12) → ≈ 3,6

σ² (variance) : =VAR.P(A1:A12) → ≈ 13,1

★
Interprétation : La moyenne (12,83) est proche de la médiane (12,5) → la distribution est assez symétrique. L'écart type de 3,6 indique une dispersion modérée (la moitié des notes est dans l'intervalle [x̄ − σ ; x̄ + σ] = [9,2 ; 16,4]).
07

Pièges fréquents & astuces

; pas ,
En Excel français les arguments se séparent avec ; (point-virgule). Exemple : QUARTILE(A1:A20;1) et non pas QUARTILE(A1:A20,1).
Cellules vides
NB() ignore les cellules vides et texte. Si tu as du texte dans ta colonne, utilise NB.SI(A1:A20;"<>"&"") pour compter les non-vides.
PEARSON vs STANDARD
Toujours utiliser ECARTYPE.PEARSON en seconde (divise par n). ECARTYPE.STANDARD divise par n−1 (pour statistiques avancées / STT).
Nommer une plage
Sélectionne A1:A20 → zone de nom en haut à gauche → tape Notes → Entrée. Tu peux ensuite écrire =MOYENNE(Notes).
Arrondir l'affichage
Pour 2 décimales sans changer la valeur : Ctrl+1 → Nombre → 2 décimales. Ou utilise =ARRONDI(MOYENNE(A1:A20);2).
Résumé statistique rapide
Dans une cellule vide tape =MOYENNE(A1:A20)&" ± "&ECARTYPE.PEARSON(A1:A20) pour afficher "12.5 ± 3.2" directement.
✓
Récapitulatif des 5 formules à maîtriser absolument :
=MOYENNE() · =MEDIANE() · =ECARTYPE.PEARSON() · =QUARTILE(;1) · =QUARTILE(;3)
